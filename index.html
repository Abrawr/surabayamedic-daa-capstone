Dijkstra — min-heap priority queue. Node jarak terkecil diproses lebih dulu. Optimal untuk bobot non-negatif.
O((V+E) log V) Optimal
Bellman-Ford — relaksasi semua edge V-1 kali. Lebih lambat tapi handle bobot negatif & deteksi siklus negatif.
O(V·E) Robust
Kruskal MST — sort edge + Union-Find. Jaringan minimum menghubungkan semua faskes.
O(E log E) MST output
